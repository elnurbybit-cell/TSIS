# TSIS3 Racer Game

This project is a Racer game written in Python using Pygame.

## Features

- Main Menu screen
- Username input
- Player car movement
- Traffic cars
- Road obstacles
- Dynamic road events
- Difficulty scaling
- Score counter
- Health system
- Power-ups:
  - Nitro
  - Shield
  - Repair
- Leaderboard saved in leaderboard.json
- Settings saved in settings.json
- Game Over screen
- Leaderboard screen
- Settings screen
