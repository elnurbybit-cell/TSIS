# TSIS2 Paint App

This project is a Paint application written in Python using Pygame.

## Features

- Pencil freehand tool
- Straight line tool with live preview
- Brush sizes: 2, 5, 10
- Rectangle
- Square
- Circle
- Right triangle
- Equilateral triangle
- Rhombus
- Eraser
- Flood-fill tool
- Text tool with live typing
- Ctrl+S saves canvas as timestamped PNG
