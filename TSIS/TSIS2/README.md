# TSIS2 Paint App

This project is a Paint application written in Python using Pygame.

## Features

- Pencil tool
- Straight line tool with preview
- Rectangle tool
- Circle tool
- Eraser tool
- Flood fill tool
- Text tool
- Brush sizes: 2, 5, 10
- Color selection
- Save canvas as PNG using Ctrl+S
