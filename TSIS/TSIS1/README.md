# TSIS1 PhoneBook

This project is an extended PhoneBook application.

## Features

- Add contact
- Show contacts
- Search contacts
- Update contact
- Delete contact
- Multiple phones per contact
- Groups
- Email
- Birthday
- Move contact to group
- CSV import
- JSON export
- Pagination
- Sorting
- PostgreSQL-ready structure
